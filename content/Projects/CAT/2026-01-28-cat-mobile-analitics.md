## Track (треккер моб приложения)
Клики по треккеру, который сохраняет атрибуцию внутри базы, для последующей загрузки ее в мобильное приложение.

Флоу: ссылка на сайте -> треккера аффилки -> треккер моб приложения -> файл apk -> внутри apk извлечение атрибуции из треккера моб приложения



- timestamp: 
- ip: 
- host: на каком домене был клик
- params: данные атрибуции, которые будут сохранены в привязке к этому IP
- product: с каким продуктом связан редирект
- useragent: какой был UA бразуереа
- geo
- mirrorUrl: на какой APK мы отправили человека
## Open
Заполняется при открытии приложения:

- timestmap
- ip
- appversion: версия приложения
- appsflyerid: не используется (убарли для уменьшения размера)
- platform: всегда андроид
- current locale: локаль телефона
- custom data: это доп информация, для атрибуии по appsflyer - игнорируем
- device data: 
- appid: это ид приложения
- deviceid: уникальный идентификатор инсталяции (если переустановить приложение он изменится)
- url: какой урл был открыт внутри
- retry: какая это попытка отркрытия url
- fcmid: фаербейс пуш токен (для отправки пушей на телефон)
- product: какой продукт
- mirrorUrl: какой было получен mirror для запуска от треккера изначально
