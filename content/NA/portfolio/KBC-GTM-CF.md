## Какие задачи перед нами ставились / Какие проблемы решали :
- сбор метрик аналитики по продукту работающему по WhiteLabel лицензии без доступа к коду продукта
- интеграция треккинга и партнерки в WhiteLabel продукт
- сбор и анализ данных по метрикам внтури продукта
- основная проблема это отсутвие доступа к коду сайта и отсутвие возможности вносить какие-либо изменения в сайт.

## Как реализовали:
Реализован скрипт для перехвата событий внтури клиенсткого приложения. Скрипт вставляется через GTM, часть логики вынесенна в CloudFlare Workers работающие поверх самого приложения. Скрипт будучи запущенным устанавливает Proxy на несколько методов фреймоврка приложения и перехватывает события и данные приложения не влияя на работу приложения. Перехваченные данные отправляются в Google Tag Manager откуда отпавляются обычными тегами в аналитику, либо по событиям происходит запуск вложенной логики внтури воркеров CF.
